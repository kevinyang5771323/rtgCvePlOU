# 会员管理系统 java542

## 项目概述

会员管理系统是基于前后端分离的架构模式，采用Spring Boot、Vue.js等现代技术开发，集成了Redis进行高效缓存处理。本项目致力于提供一套稳定、高效的会员管理解决方案。

## 技术栈

- 后端：Spring Boot、Spring、Spring MVC、MyBatis
- 前端：Vue.js
- 缓存：Redis

## 功能模块

- **首页统计**
- **产品管理**
  - 产品分类
  - 产品中心
- **会员管理**
  - 会员信息
  - 会员等级
  - 等级任务
  - 用户账单
  - 在线会员
- **订单管理**
  - 订单处理
  - 物流快递
- **电子券管理**
  - 优惠券制作
  - 已发布优惠券
  - 优惠券领取记录
- **积分管理**
- **系统管理**
  - 用户管理
  - 菜单管理
- **系统监控**
  - 操作日志
  - 异常日志
- **系统工具**
  - 定时任务
  - 存储管理
  - 邮件工具

## 系统角色

- 系统管理员
- 会员用户
- 产品管理员
- 订单管理员

## 运行环境

- Maven 3.6.3
- JDK 1.8
- Redis 3.0
- MySQL 8.0
- Vue.js 2
- Node.js 14.15

## 部署步骤

1. 环境准备：确保所有依赖环境已正确安装。
2. 数据库搭建：导入SQL文件以构建数据库结构。
3. 后端部署：通过Maven构建项目，并运行主程序。
4. 前端部署：使用Node.js构建Vue项目，并在浏览器中访问前端页面。
